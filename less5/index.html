GIT flow

GIT clone - для того, щоб "склонити" репозитори

GIT pull - для того, щоб отримати зміни з репозитора

GIT checkout-b brachname - створити нову гілку і перепідключитися на неї

GIT add. - додати зміни, котрі створили локально

GIT commit-m "commit text" - підписати їх

GIT push origin branchname - відправити на git сервер
